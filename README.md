# Cloud-Based Learning Management System (LMS)

## Project Overview

This project is a Cloud-Based Learning Management System developed using Flask and deployed on AWS.

## Features

- User Registration & Login
- Password Hashing
- Role-Based Authentication (Admin & Student)
- Dashboard
- Course Management (CRUD)
- Course Enrollment
- My Courses
- Upload Materials to Amazon S3
- Download Materials from Amazon S3
- Course Search
- Student Profile
- Admin Panel
- Flash Messages
- Custom 403 & 404 Pages

## Technologies Used

- Python
- Flask
- MariaDB
- Amazon EC2
- Amazon S3
- IAM
- Bootstrap 5
- PyMySQL
- Boto3

## Installation

```bash
git clone <repository-url>
cd LMS_Project
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
